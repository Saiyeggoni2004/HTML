HTML entities are special codes used in HTML to **display characters that are difficult or impossible to type directly from a keyboard, or to display characters that have a special meaning in HTML** (like the less than `<` or greater than `>` symbols). Here's a detailed explanation with examples: ### What are HTML Entities? When you want to display certain symbols or characters on a webpage that are not standard keyboard inputs, or if those characters are part of HTML syntax itself (like `<` or `>`), you use HTML entities. If you type a symbol like `<` directly, the browser might interpret it as the start of an HTML tag, not as a character to be displayed. HTML entities resolve this issue by allowing you to display these special characters without causing conflicts with HTML parsing. ### Why are HTML Entities Used? * **To display reserved HTML characters:** Characters like `<` (less than), `>` (greater than), or `"` (double quotation mark) have special meanings in HTML. If you want to show these characters as content, you need to use entities so the browser doesn't misinterpret them as part of the HTML structure. * **To display symbols not found on a keyboard:** Many symbols, such as copyright `©`, registered `®`, trademark `™`, or currency symbols (like the Indian Rupee `₹`), are not easily typed using a standard keyboard. Entities provide a way to include these on a webpage. * **To control spacing and formatting:** Entities like ` ` (non-breaking space) are used to add multiple spaces where a browser would normally ignore them, or to prevent a line break at a specific point. ### How to Use HTML Entities You can use HTML entities in two main ways: 1. **Using an Entity Name:** This involves using a mnemonic name followed by a semicolon (e.g., `&name;`). 2. **Using an Entity Number/Code:** This involves using a numeric code (decimal or hexadecimal) preceded by an ampersand and hash (e.g., `&#number;` or `&#xhexcode;`). Both methods achieve the same result. ### Examples of HTML Entities: * **Less Than Symbol (`<`) and Greater Than Symbol (`>`):** * To display a **less than symbol**, you would use `<` or `<`. * To display a **greater than symbol**, you would use `>` or `>`. * *Example:* If you want to show "The answer is < 10", you would write: ```html
